# Urea Validation Thresholds

Default thresholds for validating urea concentration data.

## Usage

``` r
urea_thresholds
```

## Format

A named list with thresholds for sweat and urine urea

## References

Shirreffs SM, Maughan RJ. (1997). Whole body sweat collection in humans.
J Appl Physiol, 82(1), 336-341. Lemon PW. (1991). Effect of exercise on
protein requirements. J Sports Sci.
