# Oxidation Study Class

Main container S7 class that holds all data components for an oxidation
study.

## Arguments

- calorimetry:

  A CalorimetryData object

- isotopes:

  An IsotopeData object (optional, default NULL)

- urea:

  A UreaData object (optional, default NULL)

- environment:

  An EnvironmentData object (optional, default NULL)

- subjects:

  A SubjectData object (optional, default NULL)

- protocols:

  Factor of protocol/condition names (optional, default NULL)

## Value

An OxidationStudy S7 object
