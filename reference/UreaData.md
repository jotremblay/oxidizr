# Urea Data Class

S7 class for storing urea concentration data for protein oxidation
estimation.

## Arguments

- data:

  Data frame with urea concentrations

- id_col:

  Name of the subject ID column

- sweat_urea_col:

  Name of the sweat urea concentration column (mmol/L)

- urine_urea_col:

  Name of the urine urea concentration column (mmol/L)

- urine_vol_col:

  Name of the urine volume column (mL)

## Value

A UreaData S7 object
