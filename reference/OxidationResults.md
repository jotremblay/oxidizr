# Oxidation Results Class

S7 class for storing the results of oxidation analysis.

## Arguments

- oxidation_rates:

  Data frame with oxidation rates (g/min)

- energy_contributions:

  Data frame with energy contributions (kcal and %)

- partition:

  Data frame with CHO partitioning (exo/endo/mus/liv)

- study:

  The original OxidationStudy object

- settings:

  List of analysis settings used

## Value

An OxidationResults S7 object
