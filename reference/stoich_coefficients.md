# Stoichiometric coefficients

Coefficients for calculating substrate oxidation from gas exchange, as
derived and presented in Telmosse (2022).

## Usage

``` r
stoich_coefficients
```

## Format

A named list with coefficients for CHO and fat calculations

## References
