# Isotope Validation Thresholds

Default thresholds for validating 13C isotope enrichment data.

## Usage

``` r
isotope_thresholds
```

## Format

A named list with thresholds for each isotope measurement

## References

Craig H. (1957). Isotopic standards for carbon and oxygen and correction
factors for mass-spectrometric analysis. Geochimica et Cosmochimica
Acta. Pirnay F, et al. (1977). Fate of exogenous glucose during exercise
of different intensities in humans. J Appl Physiol.
