# Subject Data Class

S7 class for storing subject characteristics and daily measurements.

## Arguments

- data:

  Data frame with subject information

- id_col:

  Name of the subject ID column

- protocol_col:

  Name of the protocol/condition column (optional)

- mass_cols:

  Named list of mass columns (initial, final)

- fluid_cols:

  Named list of fluid columns (water, saline, urine)

## Value

A SubjectData S7 object
