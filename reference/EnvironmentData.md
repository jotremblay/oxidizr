# Environment Data Class

S7 class for storing environmental conditions during testing.

## Arguments

- data:

  Data frame with environmental measurements

- id_col:

  Name of the subject ID column

- temp_col:

  Name of the temperature column (Celsius)

- humidity_col:

  Name of the relative humidity column (%)

- pressure_col:

  Name of the atmospheric pressure column (mmHg)

## Value

An EnvironmentData S7 object
