# Oxidation Summary Table

Create a summary table of oxidation rates.

## Usage

``` r
tbl_oxidation_summary(results, by = NULL, vars = NULL, decimals = 2)
```

## Arguments

- results:

  An OxidationResults object

- by:

  Grouping variable (e.g., "protocol")

- vars:

  Variables to include (NULL = auto)

- decimals:

  Number of decimal places

## Value

A gt table object
