# Filter Calorimetry Data by Time

Filter calorimetry data to a specific time range.

## Usage

``` r
filter_time_range(calo, time_range)
```

## Arguments

- calo:

  A CalorimetryData object

- time_range:

  Numeric vector of length 2 specifying min and max time

## Value

A CalorimetryData object with filtered data
